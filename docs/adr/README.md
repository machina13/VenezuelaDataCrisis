# Architecture Decision Records (ADR)

Esta carpeta registra las decisiones de arquitectura del proyecto: el contexto en
que se tomaron, la decisión, sus consecuencias y las alternativas descartadas.

Cada ADR es inmutable una vez aceptada. Si una decisión cambia, se crea una ADR
nueva que la reemplaza (campo `Reemplaza a`), no se reescribe la anterior.
